# Решение проблем с Cloudflare Pages

## Проблема: URL не открывается

### Возможные причины:

1. **Деплой еще не завершился**
   - Подождите 2-3 минуты после загрузки
   - Обновите страницу в Cloudflare Dashboard

2. **Файлы загружены неправильно**
   - Нужно загрузить файлы ИЗ папки dist, а не саму папку
   - Структура должна быть: index.html в корне

3. **Проблема с настройками проекта**

## Решение:

### Шаг 1: Проверьте проект в Cloudflare

1. Откройте Cloudflare Dashboard
2. Перейдите в "Workers & Pages" → "Pages"
3. Найдите ваш проект "myfinance-9zd"
4. Откройте его
5. Перейдите во вкладку "Deployments"
6. Проверьте статус последнего деплоя:
   - ✅ "Success" - все хорошо
   - ❌ "Failed" - есть ошибка
   - ⏳ "Building" - еще идет сборка

### Шаг 2: Проверьте структуру файлов

В проекте должны быть:
- `index.html` (в корне)
- `vite.svg`
- Папка `assets/` с файлами:
  - `index-*.js`
  - `index-*.css`

### Шаг 3: Перезагрузите файлы (если нужно)

Если деплой не удался:

1. В Cloudflare Pages откройте ваш проект
2. Перейдите в "Settings" → "Builds & deployments"
3. Или создайте новый деплой:
   - Нажмите "Create deployment"
   - Выберите "Upload assets"
   - Загрузите файлы из папки `dist` заново

### Шаг 4: Проверьте URL

Попробуйте открыть:
- `https://myfinance-9zd.pages.dev`
- `https://myfinance-9zd.pages.dev/index.html`

### Шаг 5: Проверьте консоль браузера

1. Откройте URL в браузере
2. Нажмите F12 (или Cmd+Option+I на Mac)
3. Перейдите во вкладку "Console"
4. Посмотрите, есть ли ошибки

## Альтернатива: Проверьте локально

Перед загрузкой на Cloudflare можно проверить локально:

```bash
cd /Users/mansurusupov/tg-finance-miniapp/web
npm run build
npm run preview
```

Это запустит локальный сервер для проверки собранных файлов.
