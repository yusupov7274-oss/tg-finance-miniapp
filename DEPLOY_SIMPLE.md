# Простая инструкция по деплою

## Вариант 1: Сделать репозиторий публичным (рекомендуется)

1. Откройте: https://github.com/yusupov7274-oss/tg-finance-miniapp/settings
2. Прокрутите вниз до раздела **Danger Zone**
3. Нажмите **Change visibility** → **Make public**
4. Подтвердите действие
5. Теперь откройте: https://github.com/yusupov7274-oss/tg-finance-miniapp/settings/pages
6. В разделе **Source** выберите: **GitHub Actions**
7. Нажмите **Save**

## Вариант 2: Деплой через ветку (для приватных репозиториев)

Если хотите оставить репозиторий приватным, можно использовать деплой через ветку:

1. Откройте: https://github.com/yusupov7274-oss/tg-finance-miniapp/settings/pages
2. В разделе **Source** выберите:
   - **Branch**: `gh-pages` (или `main`)
   - **Folder**: `/web/dist` (или `/ (root)` если файлы в корне)
3. Нажмите **Save**

Но для этого нужно сначала создать ветку `gh-pages` с собранными файлами.

## Рекомендация

**Лучше сделать репозиторий публичным** - это бесплатно и проще в настройке. Код веб-приложения обычно не содержит секретов, так что публичность не проблема.

После того как сделаете репозиторий публичным, GitHub Actions автоматически задеплоит сайт при каждом push в main ветку.
