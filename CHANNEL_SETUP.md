# Настройка Telegram канала для проверки подписчиков

## Шаг 1: Создание канала (если еще нет)

### Вариант A: Создать новый канал

1. Откройте Telegram
2. Нажмите на меню (три полоски) → **"Создать канал"**
3. Введите название канала (например: "Личные финансы" или "Finance App")
4. Добавьте описание (опционально)
5. Выберите тип: **"Публичный канал"** (с username) или **"Приватный канал"**
6. Нажмите **"Создать"**

### Вариант B: Использовать существующий канал

Если у вас уже есть канал, можно использовать его.

---

## Шаг 2: Добавить бота как администратора

1. Откройте ваш канал
2. Нажмите на название канала вверху
3. Выберите **"Администраторы"** или **"Управление каналом"**
4. Нажмите **"Добавить администратора"**
5. Найдите вашего бота: **@lmyfinancel_bot**
6. Выберите бота
7. **Важно:** Дайте боту хотя бы одно из прав:
   - ✅ **"Добавлять подписчиков"** (обязательно для проверки подписки)
   - Или **"Видеть участников"** (если доступно)
8. Нажмите **"Готово"** или **"Сохранить"**

**Примечание:** Бот должен быть администратором, чтобы использовать API `getChatMember` для проверки подписки.

---

## Шаг 3: Получить Channel ID

Есть несколько способов:

### Способ 1: Через бота (самый простой)

1. Добавьте бота в канал (если еще не добавили)
2. Отправьте любое сообщение в канал (или перешлите сообщение из канала боту)
3. Запустите скрипт `get-channel-id.js` (см. ниже)

### Способ 2: Через @userinfobot

1. Добавьте бота @userinfobot в ваш канал
2. Перешлите любое сообщение из канала боту @userinfobot
3. Бот покажет ID канала (будет начинаться с `-100`)

### Способ 3: Через скрипт (автоматически)

Запустите скрипт ниже — он покажет ID всех каналов, где бот является администратором.

---

## Важно!

- **Channel ID** для каналов обычно начинается с `-100` (например: `-1001234567890`)
- **Group ID** для групп начинается с `-` (например: `-123456789`)
- Сохраните этот ID — он понадобится для настройки Supabase

---

## Что делать дальше?

После получения Channel ID:
1. Сохраните его (например, в файл `.env` как `TELEGRAM_CHANNEL_ID`)
2. Переходите к Шагу 2: создание проекта Supabase
