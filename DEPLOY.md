# Инструкция по развертыванию Telegram Mini App

## Шаг 1: Сборка проекта ✅

Проект уже собран! Файлы находятся в папке `web/dist/`

## Шаг 2: Создание бота в Telegram

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Придумайте имя для бота (например: "Мои финансы")
   - Придумайте username (должен заканчиваться на `bot`, например: `my_finance_bot`)
4. **Сохраните токен бота** - он понадобится позже

## Шаг 3: Загрузка на хостинг

### Вариант A: Vercel (рекомендуется, бесплатно)

1. Зарегистрируйтесь на [vercel.com](https://vercel.com)
2. Установите Vercel CLI:
   ```bash
   npm install -g vercel
   ```
3. В папке `web` выполните:
   ```bash
   cd /Users/mansurusupov/tg-finance-miniapp/web
   vercel --prod
   ```
4. Следуйте инструкциям
5. Получите URL вида: `https://your-app.vercel.app`

### Вариант B: Netlify (бесплатно)

1. Зарегистрируйтесь на [netlify.com](https://netlify.com)
2. Перетащите папку `dist` в Netlify Drop
3. Получите URL вида: `https://random-name.netlify.app`

### Вариант C: GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите содержимое папки `dist` в репозиторий
3. Включите GitHub Pages в настройках репозитория
4. Получите URL вида: `https://username.github.io/repo-name`

### Вариант D: Свой хостинг

1. Загрузите все файлы из папки `dist` на ваш сервер
2. Убедитесь, что сервер отдает файлы по HTTPS
3. Получите URL вашего приложения

## Шаг 4: Настройка Web App в Telegram

1. Откройте **@BotFather** в Telegram
2. Отправьте команду `/newapp`
3. Выберите вашего бота из списка
4. Заполните данные:
   - **Title**: Личные финансы (или любое название)
   - **Short name**: finance (или короткое имя)
   - **Description**: Приложение для управления личными финансами
   - **Photo**: (опционально) загрузите иконку
   - **Web App URL**: вставьте URL вашего приложения (например: `https://your-app.vercel.app`)
   - **GIF**: (опционально)
   - **Short name**: finance
5. Сохраните изменения

## Шаг 5: Тестирование

1. Найдите вашего бота в Telegram
2. Откройте бота
3. Нажмите на кнопку меню (три полоски) или кнопку "Open App"
4. Приложение должно открыться!

## Важные замечания

- **HTTPS обязателен!** Telegram требует, чтобы Web App работал только по HTTPS
- Убедитесь, что все файлы доступны по указанному URL
- Если что-то не работает, проверьте консоль браузера (F12) на наличие ошибок

## Структура файлов для загрузки

Загрузите на хостинг все файлы из папки `web/dist/`:
```
dist/
  ├── index.html
  ├── assets/
  │   ├── index-*.js
  │   └── index-*.css
  └── vite.svg
```

## Полезные команды

- Сборка проекта: `npm run build`
- Просмотр локально: `npm run preview` (после сборки)
- Разработка: `npm run dev`
